GSend Pro includes a number of custom M codes that can be used within your G Code. This allows users to customize and control external elements as well as automate tasks, whilst their program runs.

The following M Codes are currently supported

M CodeDescriptionUsageRestrictions
M600Pause processing of commands for upto 2 seconds. An M600 must be immediately followed by a P value that indicates the number of milliseconds to pause for.M600P200The P value must be between 1 and 2000 milliseconds.
M605Play a sound, providing the user with an audiable warning.M605;c:\path to sound file\sound.wavThe name of the sound file to play must be included as a comment after the command on the same line
M650Add the job profile name to be used when running the codeM650 ;Cut out partsThe name of the job must appear as a comment after the command on the same line
